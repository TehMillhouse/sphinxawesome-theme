How to install the theme
========================

Depending on the use case,
there are several ways
to install the Sphinx awesome theme.
Most users should
:ref:`install the theme as a Python package <Installing the theme as a Python package>`.
If you want to modify the theme,
you may need to
:ref:`set up a development environment <Setting up a development environment>`.
